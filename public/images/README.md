# Image Placeholders

This directory should contain the following images for the website:

## Required Images

1. **hero.jpg** (1920x1080px recommended)
   - Clean, modern condo interior
   - Used in the hero section background
   - Should be high quality and professional

2. **tools.jpg** (800x600px recommended)
   - Cleaning tools or a cleaner working
   - Used in the About section
   - Professional, clean aesthetic

3. **testimonial1.jpg**, **testimonial2.jpg**, **testimonial3.jpg** (400x400px recommended)
   - Customer photos or placeholder avatars
   - Used in the Testimonials section
   - Square format works best

## Optional Images

- **service1.jpg**, **service2.jpg**, **service3.jpg**
  - Can be used to enhance service cards
  - Currently using emoji icons, but images can be added

## Image Sources

You can use:
- Unsplash (unsplash.com) - Free high-quality photos
- Pexels (pexels.com) - Free stock photos
- Your own professional photography
- Stock photo services

## Quick Setup

For development, you can use placeholder images from Unsplash:
- Hero: Search "modern condo interior"
- Tools: Search "cleaning tools professional"
- Testimonials: Search "professional headshot" or use placeholder avatar services
